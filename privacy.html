Resumen

CollabSecure es una aplicación de ejemplo (MVP) para compartir mensajes entre usuarios. Esta versión es puramente estática y almacena mensajes en el navegador (localStorage).

Medidas y prácticas aplicadas:

  • Hash de contraseñas en el cliente mediante la Web Crypto API (no se guardan contraseñas en texto plano en la sesión).
  • Validación y filtrado de contenido para evitar lenguaje ofensivo (lista básica).
  • Control de acceso: el tablero sólo aparece para usuarios autenticados (gestión en sessionStorage).
  • Los datos permanecen en el navegador del usuario y no se envían a servidores.

Advertencia: para un entorno real, implemente servidores seguros, HTTPS/TLS, almacenamiento cifrado, auditoría y políticas de retención y acceso.

Política de privacidad (resumen)

CollabSecure es una aplicación mínima para compartir mensajes entre usuarios. Esta implementación de ejemplo almacena datos en memoria y no persiste información en bases de datos.

Medidas y prácticas aplicadas:

  • Autenticación básica con contraseñas hasheadas (no se guarda en texto plano).
  • Validación y filtrado de contenido para evitar lenguaje ofensivo.
  • Los mensajes son visibles solo a usuarios autenticados.
  • Por ser un MVP de ejemplo, los datos no deben considerarse confidenciales.

Si desea probar la aplicación en un entorno real, implemente almacenamiento seguro, TLS, auditoría y políticas de retención de datos.
